RouteReady
AI-powered travel planning web app for easy travel in Pakistan.

RouteReady helps users discover destinations, build smart itineraries, find hotels, explore hidden gems, and manage travel plans with a clean, student-friendly experience.

Project Overview
RouteReady is a static frontend web application focused on travel in Pakistan.
It provides city and destination discovery with AI-style planning flows, including Islamabad-focused highlights on the homepage.

Core Features
AI-inspired travel planning flow
Destination discovery across major Pakistan cities
Featured Islamabad places on homepage
Hidden gems section with rarity-style highlights
Hotels listing page
Student mode with budget and discount support
Travel tracker and profile pages
Authentication screens (Sign In and Sign Up)
Responsive layout for desktop and mobile
Homepage Highlights
Hero: Discover Pakistan with AI
Islamabad-focused featured cards:
Faisal Mosque
Pakistan Monument
Daman-e-Koh
Margalla Hills Trail 3
Updated travel stats and localized testimonials

How To Run
Download or clone the repository.
Open the project folder in VS Code.
Launch index.html in a browser.
For best local development experience, run with a live server extension.
